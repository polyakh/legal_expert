## Features

- ⚡ Next.js 13
- React 18
- TypeScript 4
- Prettier — Code Formatter for consistent style
- 🐶 Husky — For running scripts before committing
- EditorConfig - Consistent coding styles across editors and IDEs
- Path Mapping — Import typings using the `~/` prefix
